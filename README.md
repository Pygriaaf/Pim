## **使用教程**
### **一、安装**
#### **1.如何安装**
下载之后便可直接使用。
#### **2.注意事项**
由于程序是由Python2编写，所以需要安装 [Python2](https://www.python.org/downloads/release/python-2713/) 。
### **二、命令提示符**
#### **1.正常情况下**
提示符:`$pim[文件状态]>:`
> 文件状态:
> 
> `No Files` 无文件
> 
> `No Work_Files` 无工作文件
